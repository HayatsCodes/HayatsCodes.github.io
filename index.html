About Me

Hi, I'm Hayatudeen Abdulrahman, but you can call me Hayats. I'm a Frontend Developer who enjoys building clean and responsive websites. I turn Figma designs into real, working websites using tools like React, Next.js, Tailwind CSS, and TypeScript. I've worked on different kinds of projects — from landing pages to full web apps — making sure they not only look good but also work well on all devices. My goal is always to build smooth, fast, and easy-to-use interfaces that users love. When I'm not coding, I like learning new things and helping others with web development.
